# locales ディレクトリ

日本語・英語・フランス語の翻訳をローカルで管理するための置き場です。構造は言語コードごとのサブディレクトリに `common.json` を配置しています。

```
locales/
  ja/common.json
  en/common.json
  fr/common.json
```

各 `common.json` は同じキー構造を持たせておくと差分確認や更新が容易です。翻訳追加時は以下を推奨します。

- キー追加・削除は全言語で揃える。
- UI に取り込む前にここを単一の真実源として更新し、PR でレビューする。
- 将来的に i18n ライブラリを導入する際も、このディレクトリをデータソースとして利用できます。
